Description

Universion theme features

  • Responsive and flexible
  • 6 website layouts: 1-column (content), 2-columns (content/sidebar-right, sidebar-left/content), 3-columns (content/sidebar-left/sidebar-right, sidebar-left/content/sidebar-right, sidebar-left/sidebar-right/content)
  • fixed multi-level drop down menu
  • 8 widget areas: Left Sidedar, Right Sidebar, Header Widgets, Footer Widgets, Above Content Widgets, Below Content Widgets, Above Lisp Widgets, Below List Widgets
  • Extra page template: "redirect to first subpage"
  • Support for: custom background, custom header, custom menu, custom logo, editor style, post thumbnails, microformats, rtl language support, sticky post and threaded-comments
  • Search Engines Friendly
  • Automatic rss-feeds in the head section, added rss-feed links for posts, comments, categories, tags and authors pages
  • WooCommerce Support
  • Includes a sample WordPress Child theme to speedup the development process

Installation

  1. Download file from ThemeForest and unzip it
  2. Visit plugin installation page on your website http://website.com/wp-admin/theme-install.php?upload (change 'website.com' to your website address)
  3. Choose 'universio.zip' file from this folder and press 'Install Now' button
  4. Activate 'Universio' theme
  5. Configure theme the way you want via 'Universio Settings' (http://website.com/wp-admin/themes.php?page=universio) and 'Customeze' (http://website.com/wp-admin/customize.php) pages

If you have issues with previous steps try to use FTP to upload theme to your website manually.
